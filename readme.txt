== Description ==

This is a serviceware plugin.  It gets a line of text from my server and outputs it onto your admin panel.  Meant to make everyones day a little brighter. :)

If you have suggestions for the "personality" let me know at http://blakepeterman.com/wp-personality/suggest

== Installation ==

1. Upload `personality.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
